# Tic Tac Toe – Extended Version

This project is based on the Tic Tac Toe tutorial by [Kenny Yip](https://youtu.be/LD9UEC7AI7g?si=YAFrZX7ddqQgUplt).  

## Added Features

- Score keeping for Player O and Player X.
- Three buttons for better control:
  - **Restart Game** → resets only the board.
  - **Reset Score** → resets player scores.
  - **Reset Game** → resets both board and scores.
- Automatic game restart after a win or tie.
